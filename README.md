# Systole-computation---Sage-code

This repository contains SageMath code accompanying the paper
“The systole of random hyperbolic 3-manifolds”.

- `Article_systole_computation_finte_sum.ipynb`: SageMath notebook (code)
- `Article_systole_computation_finte_sum.html`: static version with precomputed outputs

Tested with SageMath 10.2.
